What is PyCord?

A fork of discord.py. PyCord is a modern, easy to use, feature-rich, and async ready API wrapper for Discord written in Python.

Installation

Note: Python 3.8 or higher is required

  • Fresh setup:

    Just run the following command:

  • pip install py-cord

  • If discord.py is previously installed:

    discord.py is incompatible with pycord, you need to uninstall discord.py first

    To uninstall, run this command:

  • pip uninstall discord.py

    • then install PyCord with the command

  • pip install py-cord

    • Now You have sucessfully installed Pycord
